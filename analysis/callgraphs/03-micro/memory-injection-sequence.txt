    Memory 四层体系总结
    ===================

    ┌─────────────────┬─────────────────────┬──────────┬────────────────────┐
    │ 层              │ 存储位置            │ 生命周期 │ 注入方式           │
    ├─────────────────┼─────────────────────┼──────────┼────────────────────┤
    │ Auto Memory     │ ~/.claude/memory/   │ 长期     │ System Prompt      │
    │ Relevant Memory │ 同 memories/        │ 按轮召回 │ 附件消息           │
    │ Session Memory  │ 会话目录            │ 当前会话 │ postSamplingHook   │
    │ Agent Memory    │ agent 专属目录      │ agent    │ Agent Sys Prompt   │
    │ Team Memory     │ 团队同步目录        │ 团队共享 │ 合并 Auto Prompt   │
    └─────────────────┴─────────────────────┴──────────┴────────────────────┘
