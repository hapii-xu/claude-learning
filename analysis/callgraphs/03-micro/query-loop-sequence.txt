    循环终止条件
    ============

    ┌──────────────────────────────────────────────────────────┐
    │  1. 模型不输出 tool_use  → 纯文本回复，自然结束          │
    │  2. Token 预算耗尽       → checkTokenBudget() 触发       │
    │  3. 用户中断             → createUserInterruptionMessage │
    │  4. API 错误             → createAssistantAPIErrorMessage│
    │  5. 最大轮次保护         → 循环上限                      │
    └──────────────────────────────────────────────────────────┘
