    |     |              (trust 前初始化)                       |
    |     |                                                    |
    |     ├──── action ────────> setup.ts                      |

    |     |                                                    |

    |     |              (命令注册)                             |
    |     |                                                    |

    +==========================================================+
                          |
                          v
    +==========================================================+
    |  ③ 控制面 / 命令层                                       |

    +==========================================================+
                          |
                          v
    +==========================================================+
    |  ④ TUI / REPL 层                                        |
    |                                                          |

    |     +────> PromptInput（输入编排）                        |

    |             |                                            |
    |             +──> VirtualMessageList                      |

                          v

    |                                                          |
    |  QueryEngine.ts ──> query.ts                             |

数据流方向
----------

    用户输入
       |
       v
    [④ REPL / PromptInput]
       |
       v
    [⑤ query.ts] ─────> [⑥ API claude.ts]
       |                        |
       |<───── 模型响应 ────────┘
       |
       +────> [⑥ runTools / Tool.call]
       |            |
       |<── tool_result
       |
       +────> [⑥ memory 注入 / compact / hooks]
       |
       v
    [④ Messages 渲染] ──> 终端输出
