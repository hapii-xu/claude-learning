## 八、常见问题 FAQ

> **Q：为什么有这么多 hidden option？**

A：hidden option 是**内部功能或实验性功能**，不对外宣传但存在。如 `--sdk-url` 供 SDK 内部使用、`--teleport` 供远程会话桥接。它们不在 `--help` 中显示，但可以通过代码发现。

> **Q：`--bare` 与其他 option 的关系？**

A：`--bare` 是一个**总开关**，设置 `CLAUDE_CODE_SIMPLE=1`，会影响后续很多初始化行为（跳过 hooks/LSP/plugins/CLAUDE.md/keychain）。其他 option 在 `--bare` 基础上仍然生效，但部分会被忽略（如 `--chrome` 在 bare 模式无意义）。

> **Q：`--mcp-config` 和 settings 里的 MCP 配置怎么合并？**

A：默认合并（settings + `--mcp-config`）。`--strict-mcp-config` 则**只用** `--mcp-config` 的 server，忽略 settings 里的。这对于测试隔离很有用。
